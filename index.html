About the game

Radiant Tower is a browser game developed in native JavaScript and HTML Canvas.

Over the course of five days, what had meant to be a simple project became larger and larger in scope as I kept adding ideas for new monsters and different ways to code their behavior.

Every asset in the game was drawn and animated by me using a graphics tablet. You can see some of the work in progress art to the right, as well as look through the game-assets in my github repo.

I truly enjoyed working on this game. I hope to continue to add more going forward. Please feel free to contact me if you have any questions or comments.
