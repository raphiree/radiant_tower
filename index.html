About the game

Radiant Tower is a work-in-progress browser game created in native JavaScript. I plan to add more monsters and stages, as well as upgrade the art as time allows.

What was meant to be a 5-day project ended up becoming one of the most stressful and complicated projects I've ever worked on.

Every asset in the game was drawn and animated by me, and while they were extremely time-consuming, and not as detailed as I'd like them to be, watching my drawings rendered into a playable game was absolutely amazing and gratifying. You can see some of the work in progress art to the right, as well as look through the game-assets in my github repo.

I could honestly talk about this game all day. Please feel free to ask me if you have any questions.
